# Signup Tracking

Track who signs up for events!

## What's New

**SignupTracker** creates an ordered list of users who react to announcements - perfect for first-come-first-served signups.

## Key Features

- **Ordered List** - First reactor is #1, and so on
- **Real-Time** - List updates as people react
- **Deadlines** - Set automatic signup cutoffs
- **Export** - Get your list in CSV or JSON
- **History** - View past signups

## How to Sign Up

Just react to the announcement! Your position in the list is based on when you reacted.

## For Event Organizers

1. Post your announcement
2. Track it: `[p]signupset track <message_id>`
3. Users react to sign up
4. Export when ready: `[p]signupset export`

Simple and fair signup tracking!
