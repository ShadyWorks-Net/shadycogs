# Wiki Helper

Quick-access commands for server rules and info!

## What's New

**Wiki** gives moderators instant access to server rules, hosting guidelines, and helpful info commands.

## Key Features

- **Server Rules** - All 11 rules at your fingertips with `/rule <number>`
- **Hosting Info** - Link to guidelines with `/host`
- **Channel Access** - Explain genre-based system with `/noaccess`
- **Content Promotion** - Guide to Linked Roles with `/promote`
- **Server Colors** - Explain XP levels with `/colors`

## How It Works

When a member asks about rules or access:
1. Run the appropriate slash command
2. Bot posts a formatted embed with the info
3. Member gets their answer instantly

## For Moderators

- `/rule 6` - When someone posts in the wrong channel
- `/noaccess` - When someone can't see a channel
- `/promote` - When someone wants to share content
- `/hosted` - When someone asks about game servers

All prefix versions auto-delete your command message for cleaner moderation.
