# Wiki Commands

Quick-access server information!

## What's New

**Wiki** provides staff with quick commands for common questions and information.

## Features

- **Rules** - Link to specific rules
- **Game Channels** - Find the right channel for a game
- **FAQ Responses** - Canned responses for common questions
- **Reply Support** - Reply to questions with wiki info

## For Staff

Commands are configured per-server. Check your config files or ask an admin for the available commands.

Example:
```
[p]rules 3
[p]game minecraft
```

Commands auto-delete and can reply to messages!
