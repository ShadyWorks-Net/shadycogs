# Game Server Status

Check game server status right in Discord!

## What's New

**ShadyStatus** lets you query game servers to see who's online and server status.

## Supported Games

- 7 Days to Die
- ARK: Survival Evolved
- Rust
- Valheim
- Project Zomboid
- V Rising
- Any Steam A2S compatible server

## How to Use

Check all servers:
```
/serverstatus
```

Check a specific server:
```
/serverstatus server:MyServer
```

## What You'll See

- Player count (current/max)
- Server name and map
- Connection status
- Ping

Perfect for checking if your friends are online before launching the game!
