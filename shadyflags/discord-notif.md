# Smart Account Flagging

ML-powered suspicious account detection!

## What's New

**ShadyFlags** uses machine learning to identify potentially suspicious new accounts and integrates with alt tracking.

## Key Features

- **Auto-Flag** - New accounts flagged based on age
- **ML Scoring** - Risk assessment using account patterns
- **Alt Integration** - Works with ShadyAlts network data
- **Manual Flags** - Create warnings with expiry dates

## How It Works

When a new member joins:
1. Account age is checked against thresholds
2. ML extracts features (username patterns, avatar, etc.)
3. Risk score calculated
4. High-risk accounts flagged for review

## For Moderators

- Review flagged accounts in the mod queue
- Confirm or dismiss flags
- When someone is banned for botting/spam, their entire alt network is marked

Helps catch ban evaders and bot accounts before they cause problems.
