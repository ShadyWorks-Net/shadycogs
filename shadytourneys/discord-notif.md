# Tournament System

Create and manage competitive tournaments!

## What's New

**ShadyTourneys** provides a complete tournament management system with Challonge bracket integration.

## Key Features

- **Easy Signups** - `/tourney signup` to join
- **Multiple Formats** - Single elim, double elim, round robin
- **Seeding** - Automatic seeding based on rank
- **Challonge** - Auto-generated brackets
- **Match Reporting** - Report results via Discord

## How to Join a Tournament

```
/tourney signup <tournament_name>
```

Select your rank for seeding, and you're in!

## For Tournament Organizers

Create a tournament:
```
/tourney create
```

Fill in the details, and the bot handles signups, seeding, and bracket creation.

## Supported Games

Tournaments can be created for any game. Default seeding lists are available for Marvel Rivals and other games.

## Bracket Link

Get the live bracket:
```
/tourney bracket <tournament_name>
```

Good luck in your matches!
