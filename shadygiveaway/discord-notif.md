# Giveaway System

Run giveaways with automatic prize delivery!

## What's New

**ShadyGiveaway** provides a complete giveaway system with prize codes, claim verification, and automatic rerolls.

## Key Features

- **Easy Setup** - Modal-based giveaway creation
- **Prize Codes** - Automatically delivered to winners via DM
- **Claim System** - Winners must confirm to receive prize
- **Auto-Reroll** - Unclaimed prizes go to new winners
- **Multiple Winners** - Support for up to 20 winners

## How to Enter

Just click the reaction/button on the giveaway message!

## For Moderators

Create a giveaway:
```
/giveaway create
```

Fill in the prize, duration, and code. The bot handles the rest!

## Winner Flow

1. Giveaway ends
2. Winner gets DM with claim button
3. Winner clicks "Yes" to claim
4. Prize code delivered
5. If not claimed in time, new winner selected

Fair and automatic!
