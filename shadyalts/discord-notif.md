# Alt Account Tracking

Track and manage alt accounts for better moderation!

## What's New

**ShadyAlts** helps moderators track alt account networks and automatically flags related accounts when one is confirmed bad.

## Key Features

- **Alt Linking** - Manually link accounts you know are alts
- **Network View** - See entire alt networks at a glance
- **Auto-Confirmation** - When one alt is banned, all linked accounts are flagged
- **Join/Leave Alerts** - Get notified when users with linked alts join or leave

## For Moderators

### Link Alts
```
/alt mark @user1 @user2
```

### View Network
```
/alt view @user
```

### Check Suspects
```
/altnetwork suspect
```

## How It Works

1. Manually link accounts you discover are alts
2. When one account is banned for bot/spam, all linked accounts are flagged
3. Get notified when users with linked alts join or leave

Helps track ban evaders and repeat offenders across alt accounts.
