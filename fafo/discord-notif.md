# FAFO Button

A fun moderation tool with consequences!

## What's New

**FAFO** (Fuck Around Find Out) posts a tempting red button with a warning. Anyone who clicks it gets timed out.

## Key Features

- **Irresistible** - Big red button that practically begs to be clicked
- **Consequences** - Clickers receive an automatic timeout
- **Customizable** - Set your own warning message and timeout duration
- **Auto-cleanup** - Button message deletes itself after 3 minutes

## For Moderators

Run `/fafo` in any channel to deploy the button.

## Settings

- Timeout duration: 1-1440 minutes
- Custom warning message
- Configurable mod roles

Perfect for dealing with that one person who just can't help themselves.
