# In-Discord Documentation

Create a wiki right in Discord!

## What's New

**ShadyDocs** lets you create and manage documentation pages without any external hosting.

## Key Features

- **Easy Creation** - Modal-based page editor
- **Categories** - Organize pages by topic
- **Search** - Find content across all pages
- **Rich Formatting** - Embed fields, images, thumbnails
- **Autocomplete** - Page names auto-complete as you type

## How to Use

### View a Page
```
/doc <page-name>
```

### List All Pages
```
/docs
```

### Search
```
/docsearch <query>
```

## For Admins

Create pages with `/docset action:Add Page` - a modal will open for the title and content.

Perfect for server rules, FAQ, game guides, and more!
