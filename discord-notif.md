# ShadyTourneys

---

## Tournament Flow

**1. Creation** → Organizer creates tournament (solo or team)
**2. Signups** → Players click buttons to join
**3. Start** → Bracket generated on Challonge
**4. Scheduling** → Players coordinate match times in threads
**5. Matches** → Check in → Play → Report score
**6. Champion** → Winner announced with prize (if set)

---

## Match Scheduling

```
Propose Time → Accept/Counter → Check In (±15 min) → Play → Report
```

**No-show?** If opponent doesn't check in, claim forfeit.

---

## Seeding & ELO

Tournament seeding is **ELO-based**. New players start at 1000 ELO.

**Initial Seeding** (for players with no match history):

| Marvel Rivals Rank | Starting ELO |
|--------------------|--------------|
| One Above All | 1500 |
| Eternity | 1400 |
| Celestial | 1300 |
| Grandmaster | 1200 |
| Diamond | 1100 |
| Platinum | 1050 |
| Gold / Unranked | 1000 |
| Silver | 950 |
| Bronze | 900 |

After seeding, ELO adjusts naturally from match results.

- Higher ELO = higher seed in brackets
- Score margins affect ELO change
- ELO is **admin-only** (players don't see it)
