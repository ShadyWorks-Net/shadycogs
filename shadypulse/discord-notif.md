# Bot Health Monitoring

Keep an eye on bot and service health!

## What's New

**ShadyPulse** monitors the bot and connected services, alerting when things go wrong.

## Key Features

- **Health Dashboard** - Quick overview of all systems
- **Uptime Tracking** - See how long the bot has been running
- **HTTP Monitoring** - Watch external API endpoints
- **Cog Monitoring** - Detect crashes *and* command errors (with traceback snapshots), auto-reload, and alerts

## For Staff

Check bot health:
```
[p]pulse
```

## What's Monitored

- Bot latency
- External HTTP services
- Critical cogs

Status shown as: Online / Degraded / Offline
